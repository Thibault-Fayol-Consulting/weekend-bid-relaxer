# 🇬🇧 English: Weekend Bid Relaxer

## 🎯 What it does
B2B doesn't convert on Sundays. Lower targets gently on Fridays and restore on Mondays without pausing campaigns and triggering relearning.
- **Keywords/SEO:** script pause weekend google ads b2b, tcpa bid adjuster weekend script, script google ads B2B horaires

## ⚙️ Setup
1. In **Google Ads → Tools & settings → Scripts → New script**.
2. Paste the content of `main_en.gs`.
3. Set `TEST_MODE = true` for safety, then preview.

# 🇫🇷 Français : Weekend Bid Relaxer

## 🎯 Ce que fait le script
Les PME B2B ne convertissent rien le samedi/dimanche. Baissez agressivement vos cibles le vendredi sans casser l'algorithme Google.
- **Mots-clés/SEO :** script pause weekend google ads b2b, tcpa bid adjuster weekend script, script google ads B2B horaires

## ⚙️ Installation
1. Dans **Google Ads → Outils & paramètres → Scripts → Nouveau script**.
2. Collez le contenu de `main_fr.gs`.
3. Paramétrez `TEST_MODE = true` par sécurité, puis Prévisualiser.
